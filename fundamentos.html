Fundamentos de desarrollo Web HTML y CSS --------

¿Qué es el HTML?

Lenguje de Marcado que estructura el contenido de una pagina web. Basado en etiquetas que pueden ser sémanticas o no semánticas

¡No es un Lenguaje de programación!

Semántica en HTML

Etiquetas básicas del HTML

Etiqueta Descripción
h1, h2, h3, h4, h5, h6, Títulos y Subtítulos
p Párrafos
img Para mostrar las imágenes(src,alt)
ul, ol Etiquetas para definir listas desordenadas y ordenadas
Div Divisiones de contenido - dar estilos
a Es redireccionar a un link fuera o dentro de nuestra página

Formularios en HTML

Son una herramientafundamental para recopilar datos de un usuario,a través de campos como:

  • Textos
  • Casillas e texto - areas de texto
  • Listas desplegables
  • Botones

Ejemplo: Formulario de Inicio de Sesión

Enlaces a HTML

Permiten navegar a otras paginas web (a otros documentos de HTML), secciones del mismo sitio o de otra pagina. Descargar un archivo al hacer click en el enlace
